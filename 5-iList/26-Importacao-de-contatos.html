Importação de contatos externos para o iList (7)

O iList possibilita importar contatos através do preenchimento e envio de uma planilha modelo. Para que o processo seja bem sucedido, é importante seguir os passos exatamente como descritos neste artigo.

  1. Primeiramente, é preciso montar a base de dados de clientes. Como uma forma de facilitar este processo, disponibilizamos abaixo uma planilha modelo. Abra este link da Planilha Google em seu navegador:
  2. A planilha estará bloqueada para edição e você precisará realizar uma cópia para editar. Para fazer a cópia da planilha, você precisará estar logado em uma conta Google. Clique no menu “Arquivo” e, em seguida, clique em “Fazer uma cópia”.
  3. Caso você deseje gerar um arquivo Excel para salvá-lo em seu computador, clique em "Exportar para CSV"
  4. Uma tela será aberta para que possa renomear a nova planilha. No exemplo abaixo, escolhemos um nome genérico para melhor compreensão.
  5. Após realizar a cópia, existem algumas informações a serem observadas em sua planilha. Existem duas guias chamadas “Dados Contatos” e “Baixar”; iremos alterar apenas os dados da primeira guia. Não altere nenhum dado da segunda guia.
  6. Na primeira guia, vamos preencher os dados das colunas “Nome”, “E-mail” e “Telefone”, conforme nossa base de clientes. Caso você tenha alguma planilha de contatos, pode copiar e colar os dados nestas três colunas. A planilha é enviada com um exemplo, pode substituir estes dados.
  7. Na planilha, é obrigatório indicar o nome e sobrenome dos clientes. Caso algum cliente esteja com apenas um nome, será necessário preencher um segundo nome para poder realizar a importação. No exemplo abaixo, o cliente José, não seria importado para o sistema.
  8. Após preencher os dados dos seus contatos na guia “Dados Contatos”, você verá que as informações foram migradas automaticamente para a guia da planilha, intitulada “Baixar”. Esta será a base a ser enviada para o iList, com os dados já preenchidos nas colunas corretas.
  9. Agora que os dados já estão preenchidos, vamos baixar o arquivo para subir no iList. Para fazer isto, clique no menu “Arquivo”, em seguida em “Fazer download” e por último “Valores separados por vírgulas(.csv)”. Pronto! O arquivo foi baixado em seu computador.

    10. Agora vamos para a segunda etapa: subir este arquivo para o iList.

  10. Acesse o iList. Caso esteja logado no Perfil de agência, será necessário personificar o agente que será responsável por estes contatos. No menu lateral à esquerda, clique em “Leads/Contatos” e, em seguida, em “Importar Contato”.
  11. Uma nova janela será aberta. Clique em “Próximo”.
  12. No campo “Idioma”, selecione a opção “English”. Clique em “Escolher Arquivo” para inserir a planilha que baixamos há pouco. Clique em “Próximo”.
  13. Em seguida, o sistema irá exibir uma lista dos contatos que estavam na planilha e que serão inseridos no iList. Esta etapa é direcionada para a conferência de dados. Em seguida, clique em “Terminado”

Perceba que um dos contatos - João Freire - foi inserido com caracteres diferentes à acentuação padrão PT-BR. Isso fez com que seu nome fosse exibido incorretamente: ao invés de João, foi inserido como JoÃ£o. Isto pode ocorrer com nomes que contenham acentuação Nestes casos, será necessária a correção manual, posterior ao cadastro do cliente.

Pronto! Os leads foram importados com sucesso.
